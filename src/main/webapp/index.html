Available Services (JSON Format)

Op1: Create Account (/createaccount)

Request: 
{
  "input": {
    "username": "string",
    "password": "string",
    "confirmation": "string",
    "phone": "string",
    "address": "string",
    "role": "USER|BOFFICER|ADMIN"
  }
}
Response (Success): 
{
  "status": "success",
  "data": {
    "username": "string",
    "role": "USER|BOFFICER|ADMIN"
  }
}

Op2: Login (/login)

Request: 
{
  "input": {
    "username": "string",
    "password": "string"
  }
}
Response (Success): 
{
  "status": "success",
  "data": {
    "token": {
      "tokenId": "string",
      "username": "string",
      "role": "USER|ADMIN|BOFFICER",
      "issuedAt": "int",
      "expiresAt": "int"
    }
  }
}
